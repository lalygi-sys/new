# CLAUDE.md — Disko

## Что это

Disko — рабочее пространство продуктового дискавери команды IB.
Используется методология Quadruple Diamond (4 алмаза × 8 октантов).

## Структура

```
disko/
├── WELCOME.md                              — приветствие и обзор проекта
├── BACKLOG.md                              — бэклог инициатив (обновляется по мере дискавери)
├── CLAUDE.md                               — этот файл
├── artifacts/                              — артефакты каждого этапа
├── .cursor/
│   ├── rules/
│   │   └── pm-onboarding.mdc              — автозапуск опросника для PM
│   └── skills/
│       └── quadruple-diamond/
│           ├── SKILL.md                    — навык: фасилитация дискавери
│           └── references/
│               └── framework.md            — детальный фреймворк с вопросами и шаблонами
```

## Как работать

### Для AI-агента

1. При старте нового чата — **всегда** запускай сессию Quadruple Diamond (см. `.cursor/rules/pm-onboarding.mdc`)
2. Используй навык из `.cursor/skills/quadruple-diamond/SKILL.md`
3. Фреймворк с детальными вопросами и шаблонами — в `references/framework.md`
4. Обновляй `BACKLOG.md` по мере прохождения октантов
5. Сохраняй артефакты в `artifacts/`

### Режимы сбора данных

- 📋 ОПРОС — внутренние данные от PM
- 📎 ЗАГРУЗКА — существующие документы PM
- 🔍 ПОИСК — автономное исследование рынка и конкурентов
- 🧠 СИНТЕЗ — анализ и генерация на основе собранных данных

### Язык

Общение с PM — на русском (или английском, если PM предпочтёт).
Артефакты — на языке PM.
