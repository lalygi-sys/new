# PRODUCT INITIATIVE: IB Contest Platform

```
[■■■■■■■■] 8/8 — Discovery Complete
```

**North Star:** Каждый IB-партнёр может запустить прозрачный торговый контест за 5 минут
**Метрика:** Завершённые контесты/мес | 0 → 50 | 6 мес

---

## КОНТЕКСТ

IB-партнёры зарабатывают комиссию от торгового объёма клиентов. Контесты — эффективный инструмент для стимулирования торговой активности, но сегодня партнёры проводят их полностью вручную: Excel, Telegram, ручной запрос данных, ручная выплата. Процесс занимает 15-20 часов на контест, что делает их нерентабельными по трудозатратам. Ни один брокер-конкурент не предлагает self-serve решение для IB.

## ПРОБЛЕМА

**Root Cause:** Отсутствие интеграции между торговыми данными и инструментом управления контестами. IB не имеет программного доступа к результатам → всё делается вручную → процесс болезненный → контесты проводятся редко → теряется потенциал роста объёмов.

**User Impact:** IB тратит 15-20ч/контест, клиенты не доверяют результатам, споры часты, ROI неизвестен.

**Бенчмарк:** Время создания контеста: 30-60 мин → < 5 мин. Participation rate: ~5% → 15%+.

## РЕШЕНИЕ

Self-serve платформа контестов с тремя ролями:

- **IB-партнёр:** Wizard создания контеста (5 шагов, live preview), управление, результаты
- **Клиент:** Просмотр условий, accept/decline, real-time лидерборд, уведомления
- **Админ:** Dashboard мониторинга, модерация, override, дисквалификация, audit trail

**Ключевые решения:**
- Wizard с preview (паттерн Mailchimp)
- Interactive leaderboard с "You are here" (паттерн Duolingo)
- Formal accept/decline (юридическая обёртка)
- 3 типа контестов: Volume Race, Trading Streak, Top Trader
