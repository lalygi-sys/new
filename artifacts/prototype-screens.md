# Prototype Screens — IB Contest Platform

> Спецификация экранов для кликабельного прототипа.
> Каждый экран описан: что видит пользователь, ключевые элементы, действия, переходы.

---

## Навигация по ролям

```
IB Partner Portal:
  /contests              → Мои контесты (список)
  /contests/new          → Wizard создания (5 шагов)
  /contests/:id          → Детали контеста + лидерборд
  /contests/:id/results  → Итоги + выплата

Client Area:
  /contests              → Доступные контесты
  /contests/:id          → Условия + Accept/Decline
  /contests/:id/board    → Лидерборд

Admin Panel:
  /admin/contests            → Dashboard всех контестов
  /admin/contests/:id        → Детали + модерация
  /admin/contests/:id/audit  → Audit trail
```

## SCREEN MAP — переходы между экранами

```
IB PARTNER FLOW:
  1.0 (My Contests) ──→ 1.1 (Wizard Step 1)
                         ├─→ 1.2 (Step 2: Period)
                         ├─→ 1.3 (Step 3: Rules)
                         ├─→ 1.4 (Step 4: Prizes)
                         └─→ 1.5 (Step 5: Preview + Launch)
                              ├─→ 1.0 (save draft)
                              └─→ 1.6 (launched → contest detail)

CLIENT FLOW:
  2.0 (Available Contests) ──→ 2.1 (Contest Terms)
                                ├─→ 2.0 (decline)
                                └─→ 2.2 (Leaderboard — accepted)

ADMIN FLOW:
  3.0 (Dashboard) ──→ 3.1 (Contest Detail + Moderation)
                       ├─→ Disqualify participant → leaderboard recalc
                       ├─→ Pause contest
                       ├─→ Cancel contest
                       └─→ Approve payout → trigger payout

CROSS-ROLE:
  IB 1.6 (Contest Detail) ←→ Admin 3.1 (same contest, different view)
  Client 2.2 (Leaderboard) ←→ IB 1.6 (same leaderboard, IB sees all names)
```

### Цветовая схема
- Primary: Deep Blue (#1A365D) — доверие, финансы
- Accent: Gold (#D4A857) — призы, достижения
- Success: Green (#38A169) — активный статус, рост позиции
- Danger: Red (#E53E3E) — алерты, дисквалификация
- Neutral: Gray (#718096) — вторичный текст

### Типографика
- Headings: Inter Bold
- Body: Inter Regular
- Numbers/Data: JetBrains Mono (моноширинный для таблиц)

### Компоненты (переиспользуемые)
1. **Contest Card** — используется на 1.0, 2.0 (с вариациями)
2. **Leaderboard Table** — используется на 1.6, 2.2, 3.1 (с разными уровнями доступа)
3. **Progress Bar** — используется на 2.2 (клиент), 1.6 (IB)
4. **Status Badge** — Draft/Active/Completed/Flagged
5. **Wizard Step Indicator** — 1.1-1.5
6. **Alert Card** — 3.0, 3.1

### Responsive
- Desktop-first (IB и Admin)
- Mobile-friendly для Client (лидерборд, список контестов)

### Анимации (для прототипа)
- Позиция в лидерборде: slide up/down при изменении
- Wizard: slide-left при переходе между шагами
- Preview: fade-in при изменении параметров
- Notification badge: pulse при новом алерте
